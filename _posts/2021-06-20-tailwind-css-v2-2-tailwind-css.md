Tailwind CSS v2.2リリース。
JITモードの変更、CLIの書き直して`--watch`などのサポート。
JITモードで、`before`/`after`/ `first-letter`/`first-line`などの疑似要素のサポート。
また、`selection`や`mark` variantのサポート、`color:`などのtypeをつけることで柔軟な値指定ができるようになるなど
