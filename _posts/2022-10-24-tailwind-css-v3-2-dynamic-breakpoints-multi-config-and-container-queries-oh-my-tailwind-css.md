Tailwind CSS 3.2リリース。
`@config`でのconfigの読み込みをサポート。
`@supports`相当の`support-[]` variantを追加、WAI-ARIA属性に対応するvariantの追加、Data属性に対応するvariantの追加、`max-*` variantの追加。
プラグインにmatchVariant APIを追加、Container Queryに対応するプラグインの追加など
