React Native 0.71リリース。
TypeScriptがデフォルトでサポートされ、`@types/react-native`は不要となった。
Flexboxの`gap`プロパティのサポート、`aria-*` propsのサポート、PointerEventsをopt-inでサポートなど。
`propTypes`のサポートを一時的に復活、Hermesの改善など
