TypeScriptのコンパイルをRuntime時に行うts-node 10.0.0リリース。
`--script-mode`の挙動をデフォルト化し、古い挙動を再現する`--cwd-mode`を追加。
Node.js 10のサポート終了、`--show-config`の追加、`--transpiler`オプションでSWCを使ったtrasnpileに対応など。
またウェブサイトが更新されている。
