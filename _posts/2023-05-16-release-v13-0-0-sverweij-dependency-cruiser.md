モジュールの依存関係を分析するツールであるdependency-cruiser 13.0.0リリース。
Node.js 14のサポート終了、yarn 1のPnPのサポート終了、Pure ESMのパッケージに変更。
設定ファイルをESMで書けるように、`--config`で設定ファイルの渡さなくても自動で設定ファイルを探索するように変更、パフォーマンスの改善。
