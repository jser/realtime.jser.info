webpack 4.0.0βリリース。
Node.js 4のサポート終了、`mode`が必須のオプションに、`import()`は常に名前空間オブジェクトを返すように。
また、`CommonsChunkPlugin`などの一部プラグインを削除し代わりに`optimization`のオプションで設定できるように変更など
