Node.js 8.0.0リリース。
npm 5.0.0、V8 5.8、N-API(Node.js API)、`async_hooks`、WHATWG URLを正式サポート、`Buffer`コンストラクタのzero-filledをデフォルト化、`util.promisify`の追加、Node.jsが生成するErrorに`code`がつくように。
また`--debug`が廃止され`--inspect`でデバッガーと連携できるようになるなど
