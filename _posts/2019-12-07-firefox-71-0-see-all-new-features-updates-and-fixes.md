Firefox 71.0リリース。
開発者ツールのコンソールがmulti-line modeをサポート、デバッガーがInline variable previewをサポート、ネットワークのRequest Blockingをサポート。
また、Web Sockets Inspectorがデフォルトで有効化された。
Media Session APIの実装、TLS 1.0と1.1がDeprecatedになるなど
