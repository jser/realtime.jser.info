TypeScript-ESLint 4.0.0リリース。
Optional ChainingとdecoratorsのASTを変更、Scope解析を改善して`no-unused-vars`などのルールを改善、`ban-ts-comment`のデフォルト値を変更など。
またTypeScript 4.0にはTypeScript-ESLint 3.10.0で対応している。
