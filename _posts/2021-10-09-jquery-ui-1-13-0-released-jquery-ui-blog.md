jQuery UI 1.13.0リリース
jQuery 1.7以下のサポートを終了し、1.8以降の最新のバージョンを含むjQueryに対応。
このリリースをもってjQuery UIはメンテナンスモードとなり、今後はセキュリティやバグ修正のみとなる
