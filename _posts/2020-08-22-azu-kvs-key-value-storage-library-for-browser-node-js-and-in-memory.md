ブラウザとNode.jsで動作するUniversalなKey-ValueなStorageライブラリ。
同じインターフェイスをもつIndexedDBやファイルベースなどの実装があり、マイグレーションをサポートしている。
それぞれの実装は個別に分かれていて、それぞれのパッケージは1kb(gzip)程度のサイズとなっている。
