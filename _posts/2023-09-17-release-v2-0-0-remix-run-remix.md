Remix 2.0.0リリース。
React 17のサポート終了、Node.js 14/16のサポート終了。
V1からの移行は`v2_*`フラグを使うことで機能を一つずつ有効化して移行できる。
2.0.0では`v2_*`フラグの削除、`remix.config.js`の設定やコマンドの変更などが含まれる。
