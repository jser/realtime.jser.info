Node.js向けHTTPクライアントのGot 10.0.0リリース。
TypeScriptで書き直されたバージョン。
Node.js 10未満のサポート終了、`query`を`searchParams`にリネーム、`encoding`を`responseType`にリネーム、`json`オプションの変更など。
DNS cacheのサポート、`maxRedirects`オプションの追加など
